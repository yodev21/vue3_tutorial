# セットアップ

```bash
git clone https://github.com/yodev21/vue3_tutorial.git

cd vue3_tutorial

docker-compose up
```

下記 URL にアクセスし、「Welcome to Your Vue.js App」が表示されること  
http://localhost:8080/

# router リポジトリに切り替え

```bash
git checkout feature/router

docker-compose build

docker-compose up
```

下記 URL にアクセスし、画面が表示されること  
http://localhost:8080/
